# Used to Control Basic things on your Laptop or PC
Using Speech Recognition and Computer Vision

- Computer Vision is used to detect your hand, especially your finger to control the pointer / cursor and to change to Speech Command mode.
- Speech Recognition is used to command what you want to do, example " Open chrome ", etc

- Python      : 3.10.8
- NumPy       : 2.2.6
- Mediapipe   : 0.10.9
